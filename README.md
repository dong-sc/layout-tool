## 🤔 這是什麼？

排一本作品集、做一組 IG 輪播，最花時間的往往不是選照片，而是「喬版面」——
直的橫的混在一起，怎麼拼都有縫、都要裁。

這個工具幫你解決這件事：**照片丟進來，演算法自動算出剛剛好填滿版面的排法**，
不喜歡就按一下換下一種，喜歡了直接匯出。就這麼簡單。

<br>

## ⚡ 30 秒上手

```
① 拖照片進左側　→　② 勾選幾張　→　③ 按「建立頁面」
　　　　　　↓
④ 按「⟳ 換排法」直到滿意　→　⑤ 匯出 PDF / PPTX / JPEG
```

沒了，基本操作就這五步。下面是進階玩法 👇

## 🎯 六個最好用的功能

| | 功能 | 怎麼用 |
|---|---|---|
| ⭐ | **主圖** | 游標移到照片上點 ★，那張就會變成整頁最大的一格 |
| 👆 | **調整構圖** | 照片被裁到重點？**點兩下**，格子上方跳出五個定位鈕（貼左／⅓／中／⅔／貼右），點一下到位，也可以直接拖動微調 |
| 🔄 | **交換位置** | 把照片拖到另一格，兩張互換、版面自動修正 |
| ✂️ | **禁止裁切** | 一個勾勾，每張照片零裁切，整組等比縮小置中 |
| 🎨 | **自由編輯** | 每頁一顆按鈕，進入後照片隨你拖、隨你疊，附對齊磁吸 |
| 📖 | **敘事順序** | 右側面板一顆按鈕，自動生成幾種翻閱順序，⟳ 換一種，不喜歡按還原 |

## 📖 敘事順序：這幾頁該用什麼順序講？

排版解決的是「一頁怎麼擺」。但**一本作品集翻起來順不順，看的是頁與頁之間的關係**。

按右側「生成敘事順序」，工具會讀每一頁的平均明度與色調，生成幾種順序讓你挑：

| 策略 | 在做什麼 |
|---|---|
| **色彩流動** | 讓相鄰兩頁的色調跳躍最小，翻起來像色彩在慢慢過渡（實測跳躍量比明暗排序少四成） |
| **由暗到亮 / 由亮到暗** | 整本走一條明度弧線 |
| **明暗交錯** | 亮暗交替，避免一路變亮的單調感 |
| **由疏到密 / 由密到疏** | 依每頁的照片張數安排節奏 |

算出來一樣的順序會自動合併，所以候選數量會隨你的照片變動。
生成之後你還是可以手動拖動調整——**演算法只負責提案，順序最後由你決定**。

> ⚠️ 目前只依據明暗與色調。景別（特寫／中景／全景）需要真正看懂畫面內容，
> 純像素統計判斷不了，所以沒有硬做。

## 📱 IG 無縫輪播（SCRL 玩家看這裡）

選「IG 無縫輪播 4:5」、設定頁數（2–10），整組輪播就是一張連續大畫布：

- 虛線標示每一格的切分位置，每格可以**獨立換排法**
- 想要照片**橫跨兩格**？開自由編輯，把照片拖過虛線就好
- 匯出 JPEG 自動切成一組 1080×1350、照順序編號——直接上傳就是無縫接圖

<br>

## 💡 小技巧

> **一頁放 4–8 張最漂亮。**
> 只放 2–3 張時要填滿版面，裁切量天生比較大（這是數學，不是 bug 😅）。
> 真的只想放兩張又不想裁？開「禁止裁切」。

> **先設 ⭐ 再換排法。**
> 設了主圖之後，換出來的每一種排法都會保持主圖最大，比亂槍打鳥有效率。

> **換排法的前幾種通常最順眼。**
> 候選排法會先篩掉裁切太兇的，再依「相鄰照片的色彩會不會打架」重新排序，
> 色調接近的傾向被排在一起。所以前幾種就找到滿意版面的機率比較高。

> **人臉被切到，點兩下就好。**
> 滿版裁切預設取照片中段。人臉貼近邊緣被吃掉時，點兩下用定位鈕拉回來，不用換排法。

> **邊距和縫寬放心亂調。**
> 這兩個（還有版面佔比）怎麼改都不會打亂你調好的編排。

> **縫寬決定氣質。**
> `0mm` 無縫拼貼 ✦ `2–4mm` 雜誌感 ✦ `8mm+` 搭配佔比 85% = 相本裱框感

> **格子太小按不準？**
> 頂欄「預覽縮放」拉到 200%（或 Ctrl / ⌘ + 滾輪），輪播十格也能精確操作。

> **匯出永遠是原圖畫質。**
> 預覽用縮圖跑得飛快，但匯出時會回去讀原始檔重新裁切，最高 450 dpi、絕不放大插值。

## 📦 匯出格式怎麼選？

| 格式 | 適合 | 備註 |
|---|---|---|
| **PDF** | 印刷、寄給客戶 | 頁面尺寸精確到 mm，300 dpi 直接送印 |
| **PPTX** | 還想加文字的作品集 | 匯出後照片都是獨立物件，PowerPoint / Keynote 可再編輯 |
| **JPEG** | IG、混進其他文件 | 多頁自動打包 ZIP；輪播自動切格 |
