# 自動排版 — 照片自適應滿版排版工具

把一批照片丟進來，自動排成滿版版面。支援 PDF 作品集、PowerPoint、IG 貼文與無縫輪播。

**線上使用：https://dong-sc.github.io/layout-tool/**

由 [Dong Lin Photo](https://donglinphoto.com) 製作。

---

## 特色

- **自動滿版排版**：二元分割演算法依每張照片的長寬比精確解出版面，直幅橫幅混排也能拼得剛剛好，裁切極少。每頁有數十種排法可以一鍵切換。
- **完全本機處理**：照片從頭到尾不離開你的裝置，沒有上傳、沒有伺服器、沒有浮水印、免費。
- **三種輸出**：PDF（可印刷）、PPTX（可再編輯）、JPEG（單張或整組 ZIP）。
- **IG 支援**：4:5 / 1:1 / 1.91:1 尺寸與 SCRL 式無縫輪播（2–10 頁），匯出直接是 1080px 的圖檔序列。
- **可離線、可安裝**：Chrome/Edge 網址列點「安裝」即成為桌面 App，離線也能開。

## 快速開始

1. 打開網頁，把照片拖進左側照片池（JPG / PNG / WebP）
2. 點縮圖勾選要放同一頁的照片
3. 按「用選取的照片建立頁面」→ 自動排好
4. 不滿意就按「⟳ 換排法」
5. 按 PDF / PPTX / JPEG 匯出

## 功能說明

### 頁面尺寸

頂欄「頁面」下拉選擇：A4 橫／直、A3、30cm 正方（相本）、16:9 簡報、IG 各種尺寸、或自訂 mm。切換尺寸會重新排版。

### 排版控制

| 控制 | 作用 |
|---|---|
| 邊距 | 版面四周留白（mm）。改動**不會**打亂現有編排 |
| 縫寬 | 照片之間的間隔（mm）。同樣不會打亂編排 |
| 版面佔比 | 100% 滿版～60%，照片群置中縮小、四周留白。可全域設定，也可在每頁單獨覆寫 |
| 禁止裁切 | 勾選後每張照片**零裁切**：整組排版貼合照片自然比例、等比縮小置中。全域開關＋單頁覆寫 |

### 頁面內編輯

- **拖照片到另一格**＝交換兩張照片的位置，版面自動重新解算
- **拖到別頁／照片池縮圖拖進頁面**＝移動照片，兩頁都會重排
- **★（游標移到照片左上）**＝設為主圖，演算法保證它是該頁最大的一格；可設多張
- **✕（右上）**＝把照片移回照片池
- **點兩下照片**＝進入裁切構圖模式（金色外框），直接拖動照片決定顯示哪一段，點外面或 Esc 完成。匯出會套用你調整的構圖

### 自由編輯模式

每頁的「自由編輯」按鈕。進入後以目前排版為起點，照片變成自由圖層：

- 拖動＝移動位置（附頁緣、中線、其他照片邊緣的對齊輔助線與磁吸）
- 右下角把手＝等比縮放
- ⤒ ⤓＝調整疊放層級，照片可以互相覆蓋
- 匯出時按疊層順序繪製，所見即所得

再按一次退出回到自動排版；自由狀態會保留，下次進入以當下畫面重新取樣。

### IG 無縫輪播（SCRL 式）

選「IG 無縫輪播 4:5」並設定頁數（2–10）。整個輪播是一張連續大畫布，主畫布上有虛線標示每一格的切分位置，**每一格是獨立的排版區域**，各有自己的 ⟳ 換排法按鈕。

想要一張照片橫跨兩格（滑動時的接圖效果）：先讓自動排版排好 → 按「自由編輯」→ 把照片拖過虛線即可。

匯出 JPEG 會自動切成整組 1080×1350 圖檔，依序命名，直接照順序上傳 IG 就是無縫輪播。

### 頁面管理

- 右欄頁面清單：即時縮圖、點擊跳頁、**直接拖動縮圖改變頁面順序**
- 頁面標題列的 ⠿ 把手也可拖曳排序，↑↓ 按鈕微調

### 匯出設定

| 設定 | 說明 |
|---|---|
| 解析度 | 150（螢幕）／200／300（印刷標準）／450 dpi。**絕不放大**：需求超過原圖像素時自動以原圖為上限，不做無意義的插值 |
| 壓縮品質 | JPEG 品質 50–100%，控制檔案大小與畫質的取捨 |

輸出從**原始檔案**重新裁切，預覽用的縮圖只是預覽，畫質不受影響。

## 小技巧

- **先挑主圖再換排法**：設了 ★ 之後按 ⟳，換出來的每一種排法都會保持主圖最大，比盲目 shuffle 有效率。
- **一頁放 4–8 張最漂亮**：滿版模式下 2–3 張要填滿固定版面時裁切量天生較大（數學限制）；4 張以上通常能壓在 5% 以內。若一定要放 2 張又不想裁，開「禁止裁切」。
- **直橫幅混搭**：全直幅或全橫幅的頁面排法變化少，混搭時演算法能發揮的空間最大。
- **點兩下救人臉**：滿版裁切預設取照片中段，人臉貼近上緣被切到時，點兩下往下拖一點就好，不用換排法。
- **邊距縫寬放心調**：這兩項（和版面佔比）隨時改都不會打亂你調好的編排，只有換頁面尺寸或增減照片才會重排。
- **預覽縮放**（頂欄滑桿或 Ctrl/⌘＋滾輪）：十格輪播或格子很小時，放大到 200–300% 再做拖曳交換和裁切構圖，精確得多。
- **雜誌感 vs 相紙感**：縫寬 0 是無縫拼貼；2–4mm 是雜誌感；8mm 以上＋版面佔比 85% 接近相本裱框感。
- **PPTX 當中繼格式**：匯出 PPTX 後可以在 PowerPoint/Keynote 裡繼續加文字、標題、頁碼，適合作品集需要文字說明的場合。
- **JPEG 拿去混排**：如果排版頁只是你更大文件的一部分，匯出 JPEG 再插入原本的 InDesign/Word/簡報檔即可。
- **安裝成 App**：Chrome/Edge 網址列右側的安裝圖示 → 獨立視窗、Dock 圖示、離線可用。

## 隱私

所有影像處理（縮圖、排版、裁切、匯出）都在瀏覽器本機完成。這個網站沒有後端、不收集任何資料，照片不會上傳到任何地方。

## 技術備註

- 單一 HTML 檔，無建置流程；程式庫（jsPDF、PptxGenJS、JSZip）已內嵌，離線可用
- 排版核心：隨機二元分割樹＋依長寬比精確解版面，以「最大裁切量」為主的評分挑選候選排法
- PWA：service worker 快取，安裝後離線啟動
